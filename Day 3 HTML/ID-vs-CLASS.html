To target class in css we use "."
To target id in css we use "#"
tip To go to a specific section of a website you can also use id
  1. go to a website find the particular part you want to share
  2. open inspect in web broser
  3. find the section id
  4. copy it and paste it to the main link with a "#"
  5. and shre it
now whenever you will open the link you will be automatically sent to that part of the website
