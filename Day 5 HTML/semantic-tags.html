Semantic Tags

Very usefull for SEO

  1. <header> : Used to represent the top section of a web page, often containing headings, logos, and navigation.
  2. <nav> : Signifies a navigation menu on a web page.
  3. <article> : Indicates a self-contained piece of content, such as a blog post or news article.
  4. <aside> : Typically used for sidebars or content that is tangentially related to the main content.
  5. <footer> : Represents the footer of a web page, usually containing copyright information and contact details.
  6. <figure> and <figcaption > : Used for embedding images, diagrams, or charts, along with a caption.
  7. <main> : Signifies the main content area of a web page.
  8. <time> : Used to represent time-related information, like dates and times.
  9. <section> : Represents a thematic grouping of content on a web page.
